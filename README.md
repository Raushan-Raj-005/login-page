# login-page
🌐 3D Neumorphic Login Page
This is a visually appealing 3D Neumorphic Login Page built using only HTML and CSS. The design follows the popular neumorphism trend, which blends minimalism with subtle 3D effects to create soft UI components.

💡 Features
Modern Neumorphic Design: Smooth, soft-shadow elements that mimic physical surfaces.

Responsive Centered Layout: The login form is vertically and horizontally centered on the screen.

Pure HTML & CSS: No JavaScript or external libraries used—lightweight and beginner-friendly.

🧱 Built With
HTML5

CSS3 (Neumorphism Design)

🎯 Purpose
This project was created as part of a front-end development learning path (e.g., Sigma WebDev Course). It's useful for understanding:

Flexbox layout

Box shadows for 3D effects

Clean UI/UX design patterns
